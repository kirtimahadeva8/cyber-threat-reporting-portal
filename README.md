# Cyber Threat Reporting Portal

## Project Overview

Cyber Threat Reporting Portal is a web-based cybersecurity incident reporting system developed using Python Flask, MySQL, and Bootstrap 5. The system allows users to report cyber threats, upload evidence files, and track report status. Administrators can review, investigate, and manage reported incidents.

## Features

* User Registration
* User Login & Logout
* Threat Reporting System
* Evidence File Upload
* My Reports Section
* Admin Login
* Admin Dashboard
* View Report Details
* Update Threat Status
* Secure Password Hashing

## Technologies Used

* Python
* Flask
* MySQL
* Bootstrap 5
* HTML
* CSS
* Werkzeug

## Database

Database Name: cyber_threat_db

Tables:

1. users
2. threat_reports

## Installation Steps

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Create MySQL database

4. Import database.sql

5. Run the application

python app.py

6. Open browser

http://127.0.0.1:5000
